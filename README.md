# @koeroesi86/react-cv
[![Build Status](https://travis-ci.com/Koeroesi86/react-cv.svg?branch=main)](https://travis-ci.com/Koeroesi86/react-reviver)

### Dependencies
* [NodeJS](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/en/)

### Usage

```shell
yarn build && yarn start
```

Open `dist/<name>.pdf`

### Running locally

```shell
yarn dev
```

Open `dist/<name>.pdf`
